Features

Core9 gives you a pluggable framework that let's you create plugins
which are simple to use, easy to customize and extendable.

Less is more

CORE9 is developed on the JVM to write well-structured, extendable code which is easy to maintain.

Components

A collection of small, reusable components using consistent and conflict-free naming conventions.

Customizer

CORE9 CORE can be extended with features and is easy to customize to create high performance apps.

Minimal

With the plugin approach CORE9 provides a consistent experience on phones and tablets to desktops.
