2. Get Started

This chapter will be a quick intro on how te get started using CORE9 CORE. It will show you how to create a plugin. And because plugins are so central to CORE9 after this short intro you will be able to quickly build your own applications.

2.1 Gradle

Core9 makes extensive use of the gradle build system. However it is not necessary to be a gradle expert to use Core9. The opposite is true that because we have all the build logic in gradle files there is almost nothing left for the user to configure.

This becomes clear when we start a new project just download gradle and the build file and your good to go. 

$ install gradle (see gradle install)

$ mkdir plugin-logger
$ cd plugin-logger/

$ cp ../plugin-example/build.gradle .
$ cp ../plugin-example/gradle.properties .
$ cp ../plugin-example/settings.gradle .


$ vim gradle.properties

$ gradle wrapper
$ ./gradlew Eclipse (for eclipse)

$ ./gradlew --refresh-dependencies
